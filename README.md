## Synopsis

A refactoring of the ray tracing code found [here](https://github.com/hunterloftis/playfuljs-demos/tree/gh-pages/raycaster). Rewritten in TypeScript. 

Added the following features:
* Crouching
* Jumping
* Looking up & down
* Look on mouse move
